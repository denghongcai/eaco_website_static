WiFi network problems

The normal function of EACO should be based on a fluent network. Bad network may cause the transmission delay and affect the gaming experience, therefore we strongly recommend you to connect your mobile devices and PC to the same WiFi network.

If you don't have WiFi ?

  1. You may use Connectify, Wi-Fi Genius or other software to turn your PC into a WiFi hotspot.
  2. Set up a WiFi hotspot with another smartphone and connect your mobile devices and PC to it. In this way no traffic flow will be costed.

Help for Need For Speed

  1. Need For Speed may have a problem with keyboard setting. In that case, please copy the simulation files of EACO to the root directory of the game installation.
  2. If this problem is still unsolved, please make sure the keyboard setting have been restored to their default setting.
  3. User experience of gaming control differs slightly according to various versions of Need For Speed. You may choose a most suitable sensitivity of the controller.

Help for Ace Combat

  1. Ace Combat 7 demonstrated an extraordinary performance when being tested before.
  2. Please set the keyboard before starting the game. You should clear all the existing settings first and then reset keyboard mappings with your mobile phone.
  3. If any calibration problem occurs during the game, backing to the last checkpoint will fix it.
